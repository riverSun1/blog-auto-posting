# blog-auto-posting
네이버 블로그 자동 글 발행
* 셀레니움을 이용해 네이버 로그인을 동적으로 시도
* 네이버는 기계로 로그인 하는 것을 알아 차린다.
* 크롬 정보를 가져와서 네이버 로그인 우회
* pyperclip.copy 함수를 사용하여 아이디, 비밀번호 입력
* 블로그 글, 제목 작성 후 발행
